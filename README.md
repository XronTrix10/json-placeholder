## Technologies Used

 - Next.js 14
 - TypeScript
 - TailwindCSS
 - Lucide React Icons
 - jsonplaceholder REST API

## Features

 - Featured Blog Posts
 - User Login through JWT cookie
 - Create Blog Posts
 - Add & View Comments to blog
 - Modify or Delete Blogs
 - Photo Gallery
 - Author Page(s)
 - View, Modify and Delete TodoLists
